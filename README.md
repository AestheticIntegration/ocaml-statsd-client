ocaml-statsd-client
===================

[Statsd](https://github.com/etsy/statsd) client for OCaml.
Originally written by [Mike Wells](https://github.com/mwells).

Installation
------------

Requires:

* [findlib](http://projects.camlcity.org/projects/findlib.html)
  (ocamlfind command)
* [lwt](http://ocsigen.org/lwt/api/Lwt)

Build:

```
$ make
```

Install:

```
$ make install
```
